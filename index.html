Project Lombok

Project Lombok makes java a spicier language.
From letting you add getters automatically by annotating your fields with @Getter to making sure your resources get cleaned up properly and automatically by annotating your local variable declarations with @Cleanup.

Lombok currently supports javac 1.6 and Eclipse 3.4 and up.

Download lombok now!
Installation instruction: Just double-click the jar you download to start the installation process.
